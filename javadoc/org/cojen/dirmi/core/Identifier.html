      • register

        public <T> void register(T obj)
              throws IllegalArgumentException
        Register the given object with this identifier. Registered objects are not strongly referenced, and so they may be garbage collected unless referenced elsewhere. Many objects can be registered with an identifier.

        A registered object cannot be retrieved, but calling identify against a registered object returns the original identifier object.

        Throws:
        IllegalArgumentException - if given object is null
