        Returns immediately and calls listener asynchronously, but no more than once. The accept method must be called again (possibly with the same listener instance) in order to accept more sessions. If no listener is accepting incoming sessions, then the session is closed after a timeout elapses. The acceptAll method may be called at any time to switch to automatic session acceptance.
