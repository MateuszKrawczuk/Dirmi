Interface StubFactory<R extends Remote>


  • public interface StubFactory<R extends Remote>
    Produces new Stub instances for client-side Remote objects. A Stub instance marshalls requests to a remote Skeleton which in turn calls the real method. Any response is marshalled back for the Stub to decode.
    Author:
