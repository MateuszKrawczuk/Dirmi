      • inputNotify

        void inputNotify(Channel.Listener listener)
        Register a listener which is asynchronously notified when a call to isInputReady will return true. Listener is called at most once per registration.

      • outputNotify

        void outputNotify(Channel.Listener listener)
        Register a listener which is asynchronously notified when a call to isOutputReady will return true. Listener is called at most once per registration.

      • usesSelectNotification

        boolean usesSelectNotification()
        Returns true to indicate that asynchronous notifications use channel selection instead of blocked threads. For channels that aren't select-based, it can be more efficient to block on I/O streams directly rather than wait for notifications.

      • inputResume

        boolean inputResume()
        Allows input to resume after reading a suspend marker.
        Returns:
        false if not supported or if no suspend marker is present

      • isResumeSupported

        boolean isResumeSupported()

      • outputSuspend

        boolean outputSuspend()
                      throws IOException
        If supported, writes a suspend marker and flushes the output. Input side will read EOF until resumed. Input must be resumed for each suspend.
        Returns:
        false if not supported or stream is closed
        Throws:
        IOException

      • register

        void register(CloseableGroup<? super Channel> group)
        Register this channel with at least one group. When this channel is closed, it removes itself from all groups it has been registered with.

      • isClosed

        boolean isClosed()
        Returns true if channel is absolutely closed or disconnected. A return value of false doesn't imply that the next I/O operation will succeed.

      • installRecycler

        Remote installRecycler(Channel.Recycler recycler)
        Install a recycler which is called after this channel has been closed and a recyled channel has been created. Method returns an opaque remote object which allows remote channel endpoint to control this channel. Channels which don't use this feature return null.
        Throws:
        IllegalStateException - if a recycler is already installed

      • setRecycleControl

        void setRecycleControl(Remote control)
        Set the recycling control object from the remote channel endpoint. Method does nothing if channel doesn't use this feature.
        Throws:
        IllegalArgumentException - if channel uses control feature and instance is same as local control or is unsupported
