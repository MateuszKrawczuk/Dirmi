Class RecyclableSocketChannelAcceptor

  • java.lang.Object
    • org.cojen.dirmi.io.RecyclableSocketChannelAcceptor
  • All Implemented Interfaces:
    Closeable, AutoCloseable, ChannelAcceptor

    public class RecyclableSocketChannelAcceptor
extends Object
    Implements an acceptor using TCP/IP which supports channel recycling. A recycler must be installed on accepted channels, and remote endpoint must be using RecyclableSocketChannelConnector.

    Brian S O'Neill
