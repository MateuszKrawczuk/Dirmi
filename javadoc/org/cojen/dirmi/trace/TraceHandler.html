org.cojen.dirmi.trace

Interface TraceHandler

  • All Known Implementing Classes:
    NullHandler, ScopedTraceHandler, SimpleHandler

    public interface TraceHandler
    A trace handler is responsible for gathering and processing traced method invocations. Implementations must have a public constructor which accepts a TraceToolbox instance, and it may also accept a String argument. 
     public class MyHandler implements TraceHandler {
     public MyHandler(TraceToolbox toolbox, String arg) {
         ...
     }

     ...
 }
