org.cojen.dirmi.util

Class ThreadPool

  • java.lang.Object
    • java.util.concurrent.AbstractExecutorService
      • org.cojen.dirmi.util.ThreadPool
  • All Implemented Interfaces:
    Executor, ExecutorService, ScheduledExecutorService

    public class ThreadPool
extends AbstractExecutorService
implements ScheduledExecutorService
    Custom thread pool implementation which is slightly more efficient than the default JDK1.6 thread pool and also provides scheduling services. More importantly, this implementation immediately removes cancelled tasks (in O(log n) time) instead of leaking memory.
    Author:
    Brian S O'Neill
